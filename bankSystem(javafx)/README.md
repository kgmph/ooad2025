# bankingSystem(javafx)-interfaces

JavaFX GUI + clean architecture with **5 Java interfaces**:
1. `InterestBearing` (domain)
2. `AuthService` (port)
3. `AccountService` (port)
4. `AccountRepository` (repo)
5. `CustomerRepository` (repo)

Run:
```bash
mvn -q javafx:run
```
